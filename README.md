# GambleBot for Slack 
Lets go gambling >:)

About: 
Gamble bot is a bot for Slack that gives you the ability to gamble virtual currency. This bot does not condone real world gambling!

Commands:
/cf (cash) -- bet your money on a coinflip
/bj (cash) -- bet your money on blackjack
/cash -- show your cash

How to download try it:
Go on bot spam of the Hackclub "bot-spam" channel and try it out!
or
  Install for yourself:
1. Clone the repository, install nodejs, install env with '''npm install dot-env'''
2. cd into the repo and create a new .env file. In it, write 2 variables, a SLACK_BOT_TOKEN and SLACK_APP_TOKEN, in each field, write the tokens.
3. run '''node index.js to run''
